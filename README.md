# Mritunjai Chauhan Portfolio

Source for [mritunjai.dev](https://mritunjai.dev), a recruiter-facing portfolio focused on AI SaaS, realtime voice AI, campus placement systems, payments, RBAC, and edge infrastructure.

## Tech Stack

- React, TypeScript, Vite
- Tailwind CSS, Framer Motion
- React Router

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run build
npm run lint
```
